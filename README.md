# To-Do list em Python

> Esse projeto consiste em uma simples to-do list, que armazena os tarefas em arquivo json e suporta adição, remoção e atualização de status das tarefas, além de listar as tarefas atuais.

O código está sendo desenvolvido em Python e interage na manipulaçãode um arquivo json.

## Funcionalidades

- [x] Adicionar tarefas
- [x] Remover tarefas
- [x] Atualizar tarefas
- [x] Listar tarefas atuais
- [] Personalizar status da tarefa (Em breve)
- [] Listar tarefas filtradas por status (Em breve)
